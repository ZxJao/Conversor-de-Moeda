# 💱 Conversor de Moedas

Projeto de um **conversor de moedas** desenvolvido com **HTML, CSS e JavaScript**.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript

## 📁 Como usar
1. Clone ou baixe o repositório  
2. Abra o arquivo `index.html` no navegador  
3. Informe o valor e escolha as moedas para conversão  

## 🎯 Objetivo
Praticar consumo de API e manipulação do DOM com JavaScript, criando um conversor de moedas funcional e simples.

## ✨ Funcionalidades
- Conversão entre diferentes moedas  
- Atualização de valores em tempo real  
- Interface simples e responsiva
